Display a daily Bible verse in the footer of your WordPress admin area, using the Verse of the Day API.

==Description==
This plugin allows you to display a daily Bible verse in the footer of your WordPress admin area. The verse is fetched using the Verse of the Day API, and the plugin supports multiple languages.

==Features==
Displays a daily Bible verse in the footer of your WordPress admin area
Supports multiple languages (English and Spanish by default)
Caches the Bible verse for 24 hours to minimize API calls

==Installation==
Upload the plugin files to the /wp-content/plugins/bible-verses-in-footer directory, or install the plugin through the WordPress plugins screen directly.
Activate the plugin through the 'Plugins' screen in WordPress.

==Frequently Asked Questions==
Q: How can I add more languages or Bible versions?
A: Edit the supported_languages array in the plugin file and add your desired language codes (ISO 639-1) along with the corresponding Bible versions supported by the Verse of the Day API.
